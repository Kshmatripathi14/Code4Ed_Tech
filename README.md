## About

*Code4Ed_Tech* is a base template for building Streamlit-based web apps. It includes a backend component, sample data, and easy setup scripts â€” designed to help you rapidly prototype educational technology tools or dashboards.

---

## Features

- Basic Streamlit app setup that can be extended.  
- Sample data provided to test functionality early.  
- Option to run via script or directly using Streamlit CLI.  
- Clean file structure to help modular development.

| Component | Technology / Tool         |
